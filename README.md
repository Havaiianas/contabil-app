# Contábil — Frontend

App de conciliação contábil com IA, pronto para GitHub Pages.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `contabil-app`)
2. Faça upload do `index.html` para a branch `main`
3. Vá em **Settings → Pages → Source: main / root**
4. Acesse em: `https://SEU_USUARIO.github.io/contabil-app`

## Configurar IP da VPS

Edite a linha no `index.html`:
```js
const API_BASE = 'http://187.127.251.194';
```

Troque pelo IP ou domínio da sua VPS.
